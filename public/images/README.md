# Folder Images

Simpan foto-foto yang akan digunakan di website di folder ini.

## Foto yang diperlukan:

1. **hero-background.jpg** atau **hero-background.png**

   - Foto untuk background halaman utama (homepage)
   - Ukuran disarankan: 1920x1080 atau lebih besar
   - Format: JPG atau PNG

2. **login-background.jpg** atau **login-background.png**

   - Foto untuk background halaman login
   - Ukuran disarankan: 1920x1080 atau lebih besar
   - Format: JPG atau PNG

3. **logo-sekolah.png** atau **logo-sekolah.jpg**
   - Logo sekolah untuk navbar
   - Ukuran disarankan: 200x200 atau lebih besar (akan otomatis di-resize)
   - Format: PNG (disarankan untuk transparansi) atau JPG
   - Background transparan lebih baik untuk logo

## Cara menggunakan:

1. Simpan foto dengan nama yang sesuai di folder ini
2. Pastikan nama file sesuai dengan yang ada di kode:
   - `hero-background.jpg` untuk homepage
   - `login-background.jpg` untuk halaman login
3. Jika menggunakan format PNG, ubah ekstensi di kode dari `.jpg` menjadi `.png`

## Catatan:

- Foto akan otomatis di-load dari folder `public/images/`
- Pastikan ukuran file tidak terlalu besar (max 2-3 MB) untuk performa yang baik
- Gunakan format JPG untuk foto dengan banyak detail, PNG untuk foto dengan transparansi
